08 V2 → V3 CHANGES
REDESIGN Sidebar — Flat toggle list → semantic cards (Threat, Theater, Energy, Forces, Links). Each section collapsible. Data-rich instead of just layer toggles.
NEW Feed panel — Separated from sidebar into its own 300px panel. Always-visible live dot, time + side filters, click-to-fly items.
NEW Energy & Shipping section — Brent crude, Hormuz transit, Fujairah congestion, facilities struck/shutdown. Pulled from energy-tracker.py + hormuz-tracker.py data.
REDESIGN Toolbar — Grouped with visual dividers (Static | Live | Overlays). Active buttons glow with category color. Cleaner visual hierarchy.
MOVE Layer toggles — Moved from sidebar to left toolbar exclusively. Sidebar no longer has layer checkboxes.
REDESIGN Force disposition — Changed from tall stacked bars to compact inline bars in sidebar. Same data, 75% less space.
NEW Counter chips — Bottom-left quick-stat pills (fires, quakes, mil flights, live status). Always visible regardless of panel state.
REDESIGN Mobile — 5-tab bottom bar with slide-up sheets instead of full-screen panel takeover. Map always visible.
REMOVE Sidebar asset list — Long scrollable list of 200+ assets replaced by layer toggles on toolbar + click-on-map interaction.
NEW Immersive mode — Double-click map to hide all panels. Single click to restore. Keyboard: Escape.
